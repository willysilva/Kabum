

### Prerequisites

Database configurations 

```
create a database and import arquive kabum.sql 
```

alter "/core/init.php" file
```

			'username'	=> 'YOUR_DB_USERNAME',
			'password'	=> 'YOUR_DB_PASSWORD',
			'db'		=> 'YOUR_DATABASE',
```
